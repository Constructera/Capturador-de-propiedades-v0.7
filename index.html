❓ Ayuda rápida

  
Criterios de captura y uso correcto de la app.


  
1. ¿Qué es lo mínimo para una captura esencial?

  
Tipo, operación, dirección, zona, precio, m² terreno y m² construcción (ambos), fuente, quién ofrece y asesor. Para terreno también: servicios y uso de suelo. El resto puede quedar como S/I.


  
2. ¿Cómo gano las 3 estrellas?

  

    ⭐ Velocidad — Terminar en ≤ 5 minutos (activa el cronómetro antes de empezar).

    ⭐ Datos esenciales — Llenar todos los campos del punto 1.

    ⭐ Captura completa — Llenar todos los campos relevantes o marcarlos S/I / N/A explícitamente. Con 3 estrellas el estatus pasa automáticamente a "Captada" en el markdown.
  


  
3. ¿Qué diferencia hay entre S/I y N/A?

  

    S/I (Sin información) — No tienes el dato ahora pero es aplicable. Se marca como pendiente en el markdown para el agente Notion.

    N/A (No aplica) — El campo no tiene sentido para este inmueble. Ejemplo: m² construcción en un terreno sin construir. La captura no se penaliza.
  


  
4. La dirección es lo más importante

  
Es el identificador principal de cada propiedad. Escribe la dirección genérica (calle, número, colonia). El agente de Notion la geolocaliza y agrega la referencia de Google Maps. Usa 📍 para agregar coordenadas exactas desde tu celular.


  
5. ¿Y si no sé un dato?

  
Toca el botón S/I al lado del campo. Quedará marcado como pendiente, no como dato falso. Puedes completarlo después desde el Historial.


  
6. ¿Qué datos son específicos del terreno?

  

    Servicios disponibles (agua, luz, drenaje, pavimento o "Por confirmar") — esencial.

    Uso de suelo / densidad permitida — esencial; alimenta la corrida financiera.

    Estatus legal / escrituras — opcional (Título limpio, Con gravamen, Ejidal, En trámite, S/I).

    m² construcción — si no hay construcción, márcalo N/A explícitamente.

    Todos van en el campo Notas de la base 🏠 Propiedades hasta que el dueño confirme si se crean como propiedades independientes.
  


  
7. ¿Qué pasa con el código de la propiedad?

  
El código lo asigna Notion automáticamente. La app no lo genera; solo manda el markdown con los datos y Notion numera la nueva entrada. No tienes que hacer nada.


  
8. ¿Cómo mando la propiedad a Notion?

  
Genera el Markdown, cópialo y pégalo en el chat del agente de Notion. El agente crea la página en 🏠 Propiedades, vincula Zona, Contactos y Tareas, y confirma los campos creados.


  
9. Varias unidades (conjuntos)

  
Pon el número de unidades. Si todas son iguales, toca "Igualar" para copiar los datos globales a todas. Si difieren, edítalas individualmente.


  
10. ¿Cómo capturo un cliente o socio?

  
Usa la tarjeta Capturar cliente / socio del menú. Selecciona el tipo (Comprador, Propietario, Arquitecto, etc.) y aparecerán solo los campos relevantes. El markdown se envía al agente para dar de alta en la base Contactos / CRM. Puedes vincularlo después con una propiedad capturada.


  
11. ¿Se pierde lo que capturo?

  
No. Cada captura queda en el Historial de este dispositivo aunque no generes el Markdown. Puedes exportar todo el historial en JSON desde Configuración como respaldo.
